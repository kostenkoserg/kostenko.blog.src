Jenkins is leading open source automation tool that provides lot of plugins to make your CI better. Below is few of them that, on my opinion, can be helpful for any project.

>**1. [Build Monitor Plugin](https://plugins.jenkins.io/build-monitor-plugin)** - provides a highly visible view of the status of selected Jenkins jobs;

![Build Monitor Plugin](/img/2019-07-jenkins-build-monitor-plugin.png)

>**2. [Git Parameter Plugin](https://plugins.jenkins.io/git-parameter)** - adds ability to choose branches, tags or revisions from git repository configured in project;

![Git Parameter Plugin](/img/2019-07-jenkins-git-parameter-plugin.png)

>**3. [HTML Publisher Plugin](https://plugins.jenkins.io/htmlpublisher)** - publish the html reports that your build generates to the job and build pages;

![Git Parameter Plugin](/img/2019-07-jenkins-html-report-plugin.png)
