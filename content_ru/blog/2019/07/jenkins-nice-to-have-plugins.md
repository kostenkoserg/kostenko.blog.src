Jenkins - главный инструмент автоматизации с открытым исходным кодом, который предоставляет множество плагинов для улучшения вашего CI.
Ниже приведены некоторые из них, которые, на мой взгляд, станут полезным для любого проекта.

>**1. [Build Monitor Plugin](https://plugins.jenkins.io/build-monitor-plugin)** - обеспечивает весьма ощутимое представление о статусе выбранных заданий Jenkins;

![Build Monitor Plugin](/img/2019-07-jenkins-build-monitor-plugin.png)

>**2. [Git Parameter Plugin](https://plugins.jenkins.io/git-parameter)** - добавляет возможность выбирать ветки, теги или ревизии из git-репозитория, настроенного в проекте;

![Git Parameter Plugin](/img/2019-07-jenkins-git-parameter-plugin.png)

>**3. [HTML Publisher Plugin](https://plugins.jenkins.io/htmlpublisher)** - публикует HTML-отчёты, которые ваш билд сгенерировал для работы и сборки страниц;

![Git Parameter Plugin](/img/2019-07-jenkins-html-report-plugin.png)
